# Year 8 Science Lab

An interactive Year 8 (Cambridge Lower Secondary / IGCSE foundations) science course covering Biology, Chemistry and Physics.

## Features
- 12 topics, each with an illustrated guide (inline SVG diagrams), mnemonics, common exam mistakes, misconceptions, expert-debate points, real-world uses, flashcards and curated videos.
- A short quiz per topic (4 MCQ + 4 written) plus a full question bank (4 MCQ papers + 4 written papers).
- A comprehensive cross-topic "Big Exam".
- Hidden hints, instant self-assessment with model answers and links back to the guide.
- Auto-save / resume, and a stars + achievements system.

## Tech
Next.js (App Router) + TypeScript + Tailwind CSS. Progress is stored in the browser (localStorage). Deployable to Vercel.

## Develop
```bash
npm install
npm run dev
```
